Who I am

About Me

I'm a self-driven Full-Stack .NET Developer from Hyderabad, India. I graduated in 2023 with a B.Tech in Information Technology and spent my time since then building real production-grade applications — not just following tutorials.

I built MiniERP and PMS entirely from scratch, applying industry patterns like Clean Architecture, Repository Pattern, and Dependency Inversion because I understood why they exist — not just how to implement them.

I think in systems, not features. Before writing code, I think about layers, contracts, security tradeoffs, and how the system will behave at scale. That mindset is what separates production code from hobby code.

Currently deepening my knowledge in unit testing with xUnit & Moq, Docker containerization, and Azure cloud services.
